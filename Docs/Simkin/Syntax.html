Simkin Script Syntax

The general format of a method in Simkin within an XML file is: 

     <function name="MethodName" params="parameters">
     	<code statements>
     </function>

for example: 

     <function name="Print" params="s">
      System.Print(s);
     </function>
Within a TreeNode file the format is: 
     <MethodName> [ (<params>) {
     	<code statements>
     } ]

for example: 

     Print [ (s){
      System.Print(s);
     } ]
