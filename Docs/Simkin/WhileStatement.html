Simkin While Statement

This statement allows you to execute a series of other statements until an expression is true.

The format of the statement is: 

while (<test-expression>) {
 <statements...>
}

For example: 

i=0;
while (i lt 3) {
  trace(i);
  i=i+1;
}
will print: 
0
1
2
Note: you have to use the braces "{" and "}" to surround the statements after the while clause.
